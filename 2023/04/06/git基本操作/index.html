GitHub搜索技巧

  • awesome-<你想要了解的领域> : 该领域的百科大全 e.g. awesome-vue
  • <你想要了解的领域> sample : 该领域的项目案例 e.g. vue sample
  • <某领域> starter/boilerplate : 空项目模板
  • <你想要了解的领域> tutorial : 教程
