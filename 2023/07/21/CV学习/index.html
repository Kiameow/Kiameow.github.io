What Conda does

provides virtual environments for different projects, in order to avoid vesion confilct.

Conda command

  • conda env list : check all environments
  • conda create -n your_env_name python=X.X : create a new environment
  • conda activate your_env_name : enter your environment
  • conda list : check all the package downloaded via conda
  • conda install package_name[=X.X] : install [specific version of] package (not recommended)
  • conda uninstall package_name: uninstall package
  • conda deactivate: quit current environment
  • conda remove -n your_env_name –all : remove environment
